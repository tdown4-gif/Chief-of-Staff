# Daily-Use Success Criteria

Trusted External Memory v1 is usable only if capture is faster than opening a general notes app and deciding where something belongs.

## Success Criteria

- Ty can capture a thought from phone in under 5 seconds.
- Ty can search or recall that thought later with source proof.
- Ty can see what the system thinks needs attention: recent captures, active commitments, and memories needing review.
- Ty would choose this over Apple Notes for random thoughts because it asks for no folder, tag, or organization decision.

## Guardrails

- The daily home screen exposes the memory system. It is not a productivity dashboard.
- Open loops remain extracted commitment memories, not a task-management workflow.
- Do not add travel, gifts, CRM, research agents, skills, SBIR, calendar workflows, reminders, due dates, notifications, priorities, or proactive suggestions here.

## V1 Bar

The product should feel like a small app:

- Home-screen launch opens directly to capture.
- Capture is one text box and one save button.
- Save clears the box by returning to a fresh capture screen.
- Confirmation is immediate and visible.
- Raw source remains preserved for retrieval and review.
