# Founder Context

Chief of Staff exists because valuable context enters my life constantly, but my current systems do not reliably capture, connect, and resurface it when it matters.

## Core Insight

The user generates more value than they can reliably remember.

## Product Principle

The user should never have to be their own librarian.

## MVP Direction

Build Trusted External Memory first.

The MVP should focus on:

- Capture
- Recall
- Source-backed retrieval
- Open loops

The MVP should NOT focus on:

- Travel
- Gifts
- Skills
- Resume updates
- SBIR tracking
- Expense tracking
- CRM
- Research agents
- Relationship intelligence

These are future applications of the memory layer.

## North Star

If AI Memory works:

- Chief of Staff becomes possible.
- Travel recommendations become possible.
- Gift reminders become possible.
- Skills tracking becomes possible.
- Opportunity discovery becomes possible.

If AI Memory does not work:

None of those things matter.

## Success Criteria

A user can reliably answer:

- Who was that person I met?
- What idea did I have?
- What commitments did I make?
- What did I say about this topic?
- What am I forgetting?

using source-backed memory retrieval.
